## 소개
이 영상은 이전에 소개된 NPC 상인 시스템의 단점을 보완하는 방법을 설명합니다. 기존의 NPC만 사용한 상점은 플레이어가 실제로 아이템(에메랄드)을 소유하지 않아도 상점 아이템을 얻을 수 있는 문제가 있었습니다. 이를 해결하기 위해 커맨드 블록을 활용하여 조건부 거래 시스템을 구축하는 방법을 단계별로 보여줍니다.

## 주요 내용

### 1. 기존 NPC 상점의 한계점
- 플레이어의 인벤토리에 거래 아이템(에메랄드)이 없어도 상점 아이템을 얻을 수 있는 문제
- NPC만으로는 실제 거래 검증 시스템을 구현하기 어려움
- 플레이어의 양심에 의존해야 하는 문제점

### 2. 커맨드 블록을 활용한 해결책
- 레드스톤 블록을 활용한 커맨드 블록 트리거 시스템
- 조건부 명령 실행을 통한 아이템 검증
- 연쇄적 커맨드 블록 실행으로 완전한 거래 시스템 구현

### 3. 상세 커맨드 블록 설정 방법
- 임펄스, 체인, 조건부 설정의 중요성
- 올바른 커맨드 입력 방법
- 레드스톤 블록 생성 및 제거 자동화

## 실습 과정

1. **문제점 시연 및 해결책 소개 (00:00-01:20)**
   - 기존 NPC 상점 시스템의 문제점 설명
   - 커맨드 블록을 활용한 개선된 상점 시스템 소개
   - 에메랄드 10개와 1개 교환 옵션을 가진 상점 시연

2. **개선된 상점 시스템 테스트 (01:21-02:40)**
   - 에메랄드가 있을 때 정상적인 거래 확인
   - 에메랄드가 부족할 때 거래 불가능 확인
   - 기존 시스템과 개선된 시스템의 차이점 비교

3. **시스템 구현 방법 설명 (02:41-04:00)**
   - NPC 설정 및 명령어 구성 방법
   - 레드스톤 블록 생성 위치와 각 거래에 따른 배치 방법
   - 크리에이티브 모드에서 커맨드 블록 설치 과정

4. **커맨드 블록 설정 상세 해설 (04:01-끝)**
   - 임펄스 타입 커맨드 블록: 에메랄드 제거 명령어
   - 체인 타입 커맨드 블록: 아이템 지급 명령어
   - 조건부 설정의 중요성 및 연쇄 실행 원리 설명
   - 레드스톤 블록 생성 및 제거 자동화 방법
   - 서바이벌 모드에서 최종 테스트

## 자주 묻는 질문

**Q: 이 시스템은 어떤 버전의 마인크래프트에서 작동하나요?**  
A: 이 시스템은 마인크래프트 베드락 에디션과 에듀케이션 에디션에서 작동합니다. 자바 에디션에서는 명령어 구문이 다르므로 수정이 필요할 수 있습니다.

**Q: 여러 종류의 아이템을 거래할 수 있는 상점을 만들 수 있나요?**  
A: 네, 동일한 원리를 적용하여 여러 종류의 아이템 거래를 설정할 수 있습니다. 각 거래마다 별도의 NPC 대화 옵션과 그에 맞는 커맨드 블록 세트를 구성하면 됩니다.

**Q: 조건부 설정을 하지 않으면 어떻게 되나요?**  
A: 조건부 설정을 하지 않으면 첫 번째 커맨드 블록(에메랄드 제거)의 성공 여부와 상관없이 두 번째 커맨드 블록(아이템 지급)이 항상 실행됩니다. 즉, 플레이어가 에메랄드가 없어도 아이템을 받을 수 있게 되어 시스템의 의도와 다르게 작동합니다.

**Q: 레드스톤 블록을 생성하고 제거하는 과정이 왜 필요한가요?**  
A: 레드스톤 블록은 커맨드 블록을 활성화하는 트리거 역할을 합니다. NPC와 상호작용할 때마다 레드스톤 블록이 생성되어 커맨드 블록을 활성화하고, 작업이 완료되면 자동으로 제거되어 다음 거래를 위한 준비 상태로 돌아갑니다.

## 실습 코드
```
# NPC 명령어 (10개 화살 거래)
/setblock ~3 ~0 ~0 redstone_block

# NPC 명령어 (1개 화살 거래)
/setblock ~0 ~0 ~3 redstone_block

# 임펄스 커맨드 블록 (에메랄드 10개 제거)
/clear @p emerald 0 10

# 체인 커맨드 블록 (화살 10개 지급) - 조건부
/give @p arrow 10

# 체인 커맨드 블록 (레드스톤 블록 제거) - 조건부
/setblock ~3 ~0 ~0 air

# 임펄스 커맨드 블록 (에메랄드 1개 제거)
/clear @p emerald 0 1

# 체인 커맨드 블록 (화살 1개 지급) - 조건부
/give @p arrow 1

# 체인 커맨드 블록 (레드스톤 블록 제거) - 조건부
/setblock ~0 ~0 ~3 air
```
