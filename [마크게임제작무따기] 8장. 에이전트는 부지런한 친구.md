## 소개
이 영상은 길벗출판의 "마인크래프트 게임 제작 무작정 따라하기" 책에 수록된 튜토리얼 영상으로, 마인크래프트 에듀케이션 에디션에서 에이전트를 활용하여 광물을 수집하고 관리하는 방법을 설명합니다. 에이전트가 광물을 캐고, 특정 광물만 수집하는 방법, 그리고 에이전트의 인벤토리를 확인하는 방법을 배울 수 있습니다.

## 주요 내용

### 1. 에이전트의 광물 수집 기능
- 에이전트를 활용한 광물 채굴 기법
- 장애물 파괴 기능과 블록 수집 방법
- 깊이 방향으로 광물 탐색하기

### 2. 특정 광물만 수집하기
- 에이전트가 특정 광물만 찾아서 수집하도록 설정하기
- 아이템 모으기 블록 활용하기
- 철광석, 금광석 등 특정 광물 지정하기

### 3. 에이전트 인벤토리 확인
- 에이전트가 보유한 아이템 확인하기
- 인벤토리 슬롯별 아이템과 개수 파악하기
- 변수를 활용하여 슬롯 번호에 따른 아이템 정보 출력하기

## 실습 과정

1. **에이전트 광물 수집 (00:00-02:40)**
   - "광물수집" 채팅 명령어 설정하기
   - 에이전트가 장애물을 파괴하면서 아래로 이동하도록 설정
   - 모든 블록 수집하기 명령 추가
   - 광물 수집 후 플레이어에게 돌아오도록 구현
   - 수집된 광물 확인하기

2. **특정 광물만 수집하기 (02:41-04:50)**
   - 동일한 채팅 명령어 설정
   - 에이전트가 아래로 이동하며 광물 탐색
   - 아이템 모으기 블록으로 철광석과 금강석(다이아몬드)만 수집하도록 설정
   - 수집 후 플레이어에게 돌아오기
   - 특정 광물만 수집된 것 확인하기

3. **에이전트 인벤토리 확인하기 (04:51-끝)**
   - "슬롯" 채팅 명령어 설정
   - 변수를 활용해 슬롯 번호를 지정할 수 있도록 구현
   - 에이전트 인벤토리의 각 슬롯별 아이템 정보 확인
   - 블록 고유번호와 개수 정보 출력하기
   - 여러 슬롯을 순차적으로 확인하며 인벤토리 상태 파악하기

## 자주 묻는 질문

**Q: 에이전트가 수집한 광물은 어디에 저장되나요?**
A: 에이전트가 수집한 광물은 에이전트의 인벤토리에 저장됩니다. 에이전트를 우클릭하여 인벤토리를 확인할 수 있으며, 코드를 통해 슬롯별 아이템 정보를 출력할 수도 있습니다.

**Q: 특정 종류의 광물만 수집하도록 할 수 있나요?**
A: 네, 가능합니다. "아이템 모으기" 블록을 사용하여 수집하고자 하는 특정 광물(예: 철광석, 금광석, 다이아몬드 등)만 지정할 수 있습니다.

**Q: 에이전트의 인벤토리 슬롯은 몇 개까지 있나요?**
A: 에이전트의 인벤토리는 총 27개의 슬롯으로 구성되어 있습니다. 각 슬롯은 1번부터 시작하며, 각각 다른 아이템을 저장할 수 있습니다.

**Q: 에이전트가 수집한 광물을 플레이어가 가져올 수 있나요?**
A: 네, 에이전트를 우클릭하여 인벤토리를 열고 아이템을 가져올 수 있습니다. 또한 코드를 통해 에이전트가 특정 아이템을 플레이어에게 전달하도록 프로그래밍할 수도 있습니다.
