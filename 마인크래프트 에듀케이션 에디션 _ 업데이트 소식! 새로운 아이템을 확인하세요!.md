## 자주 묻는 질문

**Q: 폭죽은 어떻게 만들 수 있나요?**
A: 폭죽은 화약, 종이 등의 재료로 만들 수 있으며, 다양한 색상과 효과를 추가하려면 염료나 특수 아이템을 함께 사용해야 합니다. 정확한 제작법은 게임 내 제작 가이드에서 확인할 수 있습니다.

**Q: 배너에 여러 패턴을 어떻게 중첩할 수 있나요?**
A: 작업대에 배너를 놓고 염료와 함께 특정 아이템(예: 산딸기, 이끼 덮인 돌, 넝쿨 등)을 배치하면 패턴이 추가됩니다. 한 번 패턴이 적용된 배너에 다시 다른 패턴을 추가할 수 있어 다양한 조합이 가능합니다.

**Q: 책과 깃펜의 교육적 활용 방법은 무엇인가요?**
A: 학생들의 탐험 일지 작성, 관찰 결과 기록, 팀 프로젝트 계획서 작성, 스토리텔링 활동 등 다양한 글쓰기 활동에 활용할 수 있습니다. 작성된 책은 다른 플레이어와 공유할 수 있어 협업 활동에도 유용합니다.

**Q: 주크박스는 어떤 음악 디스크를 재생할 수 있나요?**
A: 마인크래프트에 기본적으로 포함된 음악 디스크들을 재생할 수 있으며, 각 디스크마다 고유한 음악이 저장되어 있습니다. 던전이나 보물 상자에서 발견하거나 크리퍼를 스켈레톤의 화살로 처치할 때 얻을 수 있습니다.

**Q: 개선된 제작 가이드는 모든 아이템에 적용되나요?**
A: 네, 게임 내 모든 제작 가능한 아이템에 대해 제작법을 확인할 수 있습니다. 현재 보유 중인 재료로 만들 수 있는 아이템은 녹색으로, 재료가 부족한 아이템은 빨간색으로 표시됩니다. 또한 필요한 재료와 부족한 재료도 확인할 수 있습니다.

## 실습 코드

```
# 폭죽 기본 제작법
화약 1개 + 종이 1개 = 기본 폭죽 (비행 지속시간 1)
화약 1개 + 종이 1개 + 화약 1개 = 중간 폭죽 (비행 지속시간 2)
화약 1개 + 종이 1개 + 화약 2개 = 긴 폭죽 (비행 지속시간 3)

# 폭죽 별 제작법 (폭발 효과 추가)
화약 1개 + 임의의 염료 1개 = 기본 폭죽 별 (색상 추가)
화약 1개 + 임의의 염료 1개 + 다이아몬드 1개 = 반짝이는 효과 추가
화약 1개 + 임의의 염료 1개 + 불꽃놀이 별 1개 = 모양 효과 추가

# 배너 패턴 예시
흰색 배너 + 빨간색 염료 (중앙에 배치) = 원형 패턴
흰색 배너 + 빨간색 염료 + 산딸기 = 십자 패턴
흰색 배너 + 검은색 염료 + 양모 = 테두리 패턴

# 책과 깃펜 사용법
책 1개 + 깃펜 1개 = 쓰기 가능한 책 (우클릭으로 작성)
```
