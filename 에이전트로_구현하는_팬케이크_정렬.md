## 소개

이 영상은 마인크래프트 에듀케이션 에디션에서 에이전트를 활용하여 팬케이크 정렬 알고리즘을 구현하는 방법을 보여줍니다. 팬케이크 정렬은 정렬 알고리즘 중 하나로, 팬케이크를 뒤집는 것처럼 배열의 일부를 뒤집어 정렬하는 방식입니다.

## 주요 내용

### 1. 팬케이크 정렬 알고리즘
- 팬케이크 정렬의 기본 원리 설명
- 에이전트를 활용한 구현 방법
- 실제 동작 과정 시연

### 2. 에이전트 프로그래밍
- 에이전트의 이동과 블록 조작
- 조건문과 반복문 활용
- 효율적인 알고리즘 구현
